MetaTweet: Not Only Just a Twitter Client!

MetaTweet は高度に拡張可能なクライアント/サーバモデルによるプラットフォームであり、Twitter に代表されるマイクロブログ サービスへのアクセスを提供するインテリジェント ゲートウェイです。MetaTweet はまた、独立した汎用のデータ・オブジェクトモデルも提供します。

What is MetaTweet?

MetaTweet は以下の問題を解決するはずです:

  • マシンの前に居ないときも Twitter をヲチしていたい
  • 複数台クライアントを持っていて API 残数が激減して勿体ない
  • なぜ Twitter クライアントがユーザ情報を管理しないのか理解できない
  • 普通のクライアントじゃ物足りない

MetaTweet は:

  • クライアントとサーバに分離しており、とりあえずサーバだけ動かしておけば、勝手に Twitter 上の情報を蓄積してくれます。クライアントはサーバに蓄積された情報を表示します。
  • サーバは一台用意して、そのサーバに複数箇所からアクセスすれば、ネットワークの負荷も、それよりも更に貴重な資源である API 呼び出し回数も節約できます。
  • MetaTweet は Twitter (に限りませんが…) から取得した全ての情報を蓄積します。ユーザ情報も、時系列に沿って全て保持されます。蓄積先はデータベースです。アプリケーションを終了させても、取得した蓄積データを放り投げることはしません。
  • 蓄積したデータは抽象化された形式で保存されたので、容易に解析・フィルタリングすることができます。
  • MetaTweet は標準で Web サーバ機能を持ち、Web インターフェイス経由で操作することができる他、Twitter API をシミュレートした形式で情報を出力したり、User Streams 風のリアルタイムな情報の取得も可能です。
  • MetaTweet のサーバは高度にモジュール化されており、様々な機能を追加したり、または不要な機能を削除することができます。モジュールの開発は恐らく簡単かもしれません。
  • TwitterIrcGateway というソフトウェアをご存知の方は、
    TwitterIrcGateway から IRC サーバ機能を削り、独自のデータ構造と DB バックエンドを追加し、サーバをモジュール化したゲートウェイ・システム
    という理解が可能です。将来的に IRC サーバモジュールが開発される可能性も十分あります。
  • 未だに開発中ですが、将来的にはここに書けないような恥ずかしい大風呂敷を拡げた機能拡張のロードマップも開発者の脳内に用意してあります。

MetaTweet が想定しているユーザ層:

  • Twitter に常駐しているコアユーザ
  • 古い TL の内容、会話の流れ等を確認することの多い方
  • 複数のマシンで Twitter クライアントを実行している方
  • 分析屋さん、その他 TL から情報を抽出したい方
  • ネットウォッチャー、ネットストーカー
  • 人柱、新しもの好き、冒険者、Twitter クライアント開発者
  • その他諸々

MetaTweet は使えるのか:

  • まだ完成してませんがそこそこ使えます。少なくとも遊べるとは信じています。
  • バグは多少ありますがだいたい安定しているように見えます。
  • 発言およびユーザ情報の蓄積についてはだいたい出来てる気がします。
  • インストールは比較的簡単です
    (ソフトウェアのインストール、XML ファイルの手編集、やや壊れた英語の文章を最低限読む程度の能力が前提となっておりますが、将来的にはまともになる予定です)

Get MetaTweet

MetaTweet プロジェクトは SourceForge.netでホストされており、リリース パッケージを ダウンロード ページから入手できます。
MetaTweet のソースコードは Git によって管理されており、GitHub でホストされています。

以下のコマンドで Git リポジトリをクローンできます:

git clone git://github.com/takeshik/metatweet

また、ソースコードは Zip アーカイブ や tarball としても入手できます。

Documentation

リファレンスは Microsoft HTML Help 形式 (.chm) でリリース パッケージおよびソースコード リポジトリ内に含まれています。
その他のドキュメントは RedMine プロジェクト サイトに置かれています。

Support

RedMine プロジェクト サイトでチケットを発行できます。バグ レポート、サポート、パッチの投稿などに利用可能です。

あるいは、メールや Twitter で直接連絡を取ってください。
