MetaTweet: Not Only Just a Twitter Client!

MetaTweet は高度に拡張可能なクライアント/サーバモデルによるプラットフォームであり、Twitter に代表されるマイクロブログ サービスへのアクセスを提供するインテリジェント ゲートウェイです。MetaTweet はまた、独立した汎用のデータ・オブジェクトモデルも提供します。

Get MetaTweet

MetaTweet プロジェクトは SourceForge.netでホストされており、リリース パッケージを ダウンロード ページから入手できます。
MetaTweet のソースコードは Git によって管理されており、GitHub でホストされています。

以下のコマンドで Git リポジトリをクローンできます:

git clone git://github.com/takeshik/metatweet

また、ソースコードは Zip アーカイブ や tarball としても入手できます。

Documentation

リファレンスは Microsoft HTML Help 形式 (.chm) でリリース パッケージおよびソースコード リポジトリ内に含まれています。
その他のドキュメントは RedMine プロジェクト サイトに置かれています。

Support

RedMine プロジェクト サイトでチケットを発行できます。バグ レポート、サポート、パッチの投稿などに利用可能です。
また、SourceForge.net プロジェクト サイトではフォーラムも設置されています。
あるいは、メールや Twitter で直接連絡を取ってください。
