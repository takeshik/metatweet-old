MetaTweet: Not Only Just a Twitter Client!

MetaTweet is a highly-extensible client-server modeled intelligent gateway as one platform, namely Hub System, to access micro-blogs such as Twitter, etc. MetaTweet also provides independent generic data/object model.

What is MetaTweet?

MetaTweet solves problems such as:

  • Watch Twitter's timeline anytime and anywhere, even though not in front of my PC.
  • Running many Twitter clients, and there are no API call counts.
  • Want to get all information on Twitter, including user information.
  • Ordinary Twitter clients are not satisfactory to me.

MetaTweet is:

  • Separated between client and server; if you only run server, the server stores informations on Twitter (etc.) Clients gets and show information from server.
  • Run only one server and access it via many clients, then you can reduce network load, and API call counts, which is more precious resource.
  • MetaTweet stores all information on Twitter (and other micro-blogs), and also user information chronologically, to databases. Data are saved if you shutdown the system.
  • Data are stored by abstract data format, so you can filter and analyze them easily.
  • MetaTweet have web-server feature in default, and you can manage and access the system via web, and you can get information with Twitter API-simulated format, and User Streams-like realtime fetching.
  • MetaTweet server is highly-modulized, so you can add and remove features easily. To develop modules may be easy.
  • Under development, but it should be work, and will be advanced more and more!

Get MetaTweet

MetaTweet Project is hosted by SourceForge.net, you can get release packages from the Download Page.
Source codes of MetaTweet is managed with Git, and hosted by GitHub.

You can clone the repository by the below command:

git clone git://github.com/takeshik/metatweet

And, you can also download the source codes as an zip archive / tarball.

Documentation

Documents and manuals are managed in MetaTweet wiki repository (git://github.com/takeshik/metatweet.wiki.git) of GitHub.
